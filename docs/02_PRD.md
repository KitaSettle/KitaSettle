# KitaSettle Product Requirements Document v1.0

## Product

KitaSettle Alpha

## Objective

Create a working web application that lets a founder open one dashboard and see an Executive Brief containing the most important priorities, decisions, risks, and opportunities.

## MVP Scope

### Must Have

1. Login screen
2. Dashboard screen
3. Executive Brief section
4. Top 3 priorities
5. Decisions waiting
6. Risks
7. Opportunities
8. Knowledge page placeholder
9. Premium responsive UI

### Nice To Have Later

1. Real AI generation
2. Google Drive integration
3. PDF upload and summarisation
4. Calendar integration
5. Email integration
6. Payment system
7. Multi-user teams

## Do Not Build Yet

- Mobile app
- Full AI workforce
- Complex automation
- Advanced analytics
- Admin panel
- Stripe billing
- WhatsApp bot

## Primary User Story

As a busy founder, I want to open KitaSettle each morning and immediately know what requires my attention so I feel clear, prepared, and in control.

## Success Criteria

The first user should say:

“I finally feel like I’m not running everything alone.”

## Alpha Screens

### Login

Simple premium sign-in page.

### Dashboard

Main home screen showing:
- Greeting
- Today’s date
- Executive Brief
- Top priorities
- Decisions waiting
- Risks
- Opportunities
- Quick actions

### Knowledge

Simple page showing future upload/search/summarise functionality.

## Design Style

- Premium
- Minimal
- Modern
- Calm
- Apple-like
- Dark text on light background
- Rounded cards
- Lots of whitespace
- Professional, not gimmicky
