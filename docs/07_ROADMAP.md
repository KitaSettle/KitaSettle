# KitaSettle Roadmap v1.0

## Alpha

Goal:
Build and test the core experience with Dan.

Features:
- Login
- Dashboard
- Executive Brief
- Knowledge placeholder

## Beta

Goal:
Test with 5–10 founders.

Features:

- Manual data entry
- PDF upload
- Basic AI summary
- User accounts

- Document memory

- Subscription billing
